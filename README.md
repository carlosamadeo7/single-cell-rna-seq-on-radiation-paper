# scRNAseq-Radiation-Response-in-SmoA1-Medulloblastoma
scRNA seq analysis on transgenic SmoA1 medulloblastomas 5 h (n = 4) and 24 h (n = 3) after 1 Gy of radiation, as compared with sham-treated control animals (n = 3) 

1. Data was obtained from : Therapeutic radiation drives leptomeningeal dissemination of medulloblastoma through an innate immune process: https://www.sciencedirect.com/science/article/pii/S1534580725003715
2. .RData object is available upon request from the author
3. Tumor-micro-environment scRNA seq analysis from previous Seurat Object
4.  .TME_RData object is available upon request from the author
