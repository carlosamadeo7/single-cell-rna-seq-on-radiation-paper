# scRNAseq workflow on Radiation Response in SmoA1 Medulloblastoma
scRNA seq analysis on transgenic SmoA1 medulloblastomas 5 h (n = 4) and 24 h (n = 3) after 1 Gy of radiation, as compared with sham-treated control animals (n = 3) 

1. Data was obtained from : Therapeutic radiation drives leptomeningeal dissemination of medulloblastoma through an innate immune process: https://www.sciencedirect.com/science/article/pii/S1534580725003715
2. `output` folder contains integrated workflow analysis on the 11 samples from 3 differnet conditions and downstream analysis.
3. `TME` folder contains processing and dowstream analysis on the tumor-microenvironment from the integrated object.
4. All Seurat.R objects are available upon request from the author
